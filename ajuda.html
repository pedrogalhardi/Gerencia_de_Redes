Como utilizar o programa?

Basta inserir o IP do destino e nome da comunidade e clicar em Iniciar. Assim começará com o processo de monitoramento e para encerrar basta clicar em Interromper.

Como foi feito o programa?

Este trabalho foi desenvolvido com tecnologias WEB, tais como Java Script, HTML, CSS e para o backEnd PHP..
